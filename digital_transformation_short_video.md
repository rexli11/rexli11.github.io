用途：104 履歷作品集 / GitHub Pages / 面試補充

格式建議：16:9，溫暖淺色底，深綠與金色作為數據重點  
公開原則：此專案為自有公司，可公開品牌名稱；公開頁面不放個人電話與 Email。

## 專案定位

將原始簡報轉化為「餐飲品牌數位轉型與營運優化案例」。重點呈現從傳統早餐店經營模式，轉向資料驅動、AI 輔助、LINE 私域經營、Google Ads 投放與線上訂餐流程的整合成果。

## 核心成果

- 人力成本下降 42%
- 線上外客單價達地區均值 3 倍
- 六個月內 LINE 官方帳號好友數 715+
- 線上訂餐機制估計約 70%
- 總曝光 395,000
- 總點擊 2,259
- 本地動作 5,288
- 均月支出 NT$1,225
- CPC 2.17
- CPM 13.65
- CTR 0.63%
- 客單價提升約 3 倍
- 物料成本佔比由 72% 降至 31%
- 線上訂餐比例約 65%
