# Resume Project Pages

Static GitHub Pages pages for resume project case studies.

## Privacy

The AD performance case study removes client names, competitor names, emails, ad account IDs, internal paths, and identifiable brand assets.

The digital transformation case study is based on Rex's own brand, so the brand name is public. The public page does not include personal phone or email details.

## Files

- `index.html`: project landing page
- `work-experience/index.html`: career timeline page, with one page per role
- `ad-performance/index.html`: AD performance animated case-study page
- `digital-transformation/index.html`: Odie breakfast digital transformation case-study page
- `social-dashboard/index.html`: social analytics dashboard operation case-study page
- `assets/ad_campaign_result_short_video.md`: script and storyboard
- `assets/104_project_video_canva_brief.md`: Canva and 104 resume brief
- `assets/digital-transformation/digital_transformation_short_video.md`: digital transformation script and storyboard
